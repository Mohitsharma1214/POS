# AI Podcast Guest Intelligence Frontend

## Tech Stack
- Next.js 14
- TypeScript
- Tailwind CSS
- shadcn/ui
- Framer Motion
- Recharts
- lucide-react
- Zustand

## Setup Instructions

1. **Install dependencies:**
   ```bash
   cd Frontend
   npm install
   ```
2. **Run the development server:**
   ```bash
   npm run dev
   ```
3. **Build for production:**
   ```bash
   npm run build && npm start
   ```
